# Civilization OS - 文明の再設計マニフェスト

## これは何？

現代社会の「のし上がり」構造を問い直し、文明のベースライン・デルタを起こすための思想と実践の記録。

## 核心的な問い

- なぜ価値を生む人が報われず、権力を奪う人が勝つのか？
- 「のし上がり」を免除された社会はどうあるべきか？
- 技術で文明のOSを書き換えることはできるか？
